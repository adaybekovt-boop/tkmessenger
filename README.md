# 🚀 Orbits Messenger

Децентрализованный production-ready мессенджер с поддержкой реального времени, видеозвонков и геолокации.

## ✨ Продвинутые функции

### 🛰️ Локатор (People Nearby)
- Поиск пользователей в радиусе 1-50 км.
- Высокая производительность благодаря Redis Geospatial Index.
- Режим "невидимка" для полной приватности.
- Обновление позиции в реальном времени через WebSockets.

### 📞 Discord-подобные звонки
- Видеозвонки 1-1 и групповые конференции.
- Демонстрация экрана в HD качестве.
- Управление микрофоном и камерой (Mute/Unmute).
- WebRTC сигналинг через Socket.io.

### 🎨 Кастомные темы
- **Sakura** — падающие лепестки сакуры (Canvas).
- **Matrix** — классический зеленый код на черном фоне.
- **Cyberpunk 2077** — неоновый интерфейс и глитч-эффекты.
- **Aurora** — динамическое северное сияние.
- **Retro Terminal** — стиль олдскульного CRT монитора.

### 🔋 Экономия ресурсов
- **Battery API**: автоматическое определение уровня заряда.
- Отключение тяжелых анимаций при <20% заряда.
- Адаптивное качество видео и сжатие медиа при медленном соединении.
- Оптимизация WebSocket heartbeat для экономии батареи.

## 🛠 Технологический стек
- **Frontend**: React 18, TypeScript, TailwindCSS, Zustand, Vite.
- **Backend**: Node.js, Express, Socket.io, Prisma ORM.
- **Database**: PostgreSQL (основная), Redis (кэш/гео/сокеты).
- **Infrastructure**: Docker, GitHub Actions.

## 🚀 Быстрый старт (Local)

1. Клонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Запустите инфраструктуру: `docker-compose up -d`.
4. Запустите проект:
   - Бэкенд: `npm run dev:server`
   - Фронтенд: `npm run dev:client`
